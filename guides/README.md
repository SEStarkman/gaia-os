# Guides

**Start here.** These guides walk you through the most common tasks you will do with your AI agent and OpenClaw. They are written for people who are not developers. No coding experience required.

---

## What's In This Folder

| Guide | What It Covers |
|-------|---------------|
| [Initialization Prompt](INITIALIZATION-PROMPT.md) | The template you paste as your first message to set up your agent's personality, preferences, and rules. |
| [Command Cheat Sheet](COMMAND-CHEAT-SHEET.md) | Every OpenClaw command you will actually use, organized by task. Print this out or bookmark it. |
| [How to Update OpenClaw](OPENCLAW-UPDATE-GUIDE.md) | Step-by-step instructions for updating to the latest version. Takes about 2 minutes. |
| [Secret Management](SECRET-MANAGEMENT.md) | How to safely handle API keys and passwords. The golden rule: never paste secrets into chat. |
| [Image Generation Prompts](IMAGE-GEN-PROMPTS.md) | Tips for getting great results from AI image generation. Prompt structure, style keywords, and examples. |

---

## Recommended Reading Order

If you are just getting started:

1. **[Initialization Prompt](INITIALIZATION-PROMPT.md)** first. This is how you introduce yourself to your agent.
2. **[Command Cheat Sheet](COMMAND-CHEAT-SHEET.md)** next. Keep it handy for daily use.
3. **[Secret Management](SECRET-MANAGEMENT.md)** before you connect any tools or APIs.
4. **[How to Update OpenClaw](OPENCLAW-UPDATE-GUIDE.md)** when a new version comes out.
5. **[Image Generation Prompts](IMAGE-GEN-PROMPTS.md)** whenever you want to create images.
